# Description

This API is used to fetch the commits of an specific repository, in this case going to be `this` repository.

## Dev environment

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ ng serve
```

Navigate to `http://localhost:4200/`.

## Running unit tests

```bash
# testing
$ ng test
```

## Docker

```zsh
$ docker-compose up --build
```

Navigate to `http://localhost:4000/`.
